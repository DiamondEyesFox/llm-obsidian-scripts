# Inbox

Files for Claude to process.

## Subfolders

| Folder | Purpose |
|--------|---------|
| `To Process/` | Drop files here for Claude to work on |
| `Context/` | Background info Claude should know about |
| `Requests/` | Specific asks or task descriptions |

## How to Use

1. Drop a file in `To Process/`
2. Tell Claude "check the inbox" or reference the file
3. Claude will read it and act accordingly
