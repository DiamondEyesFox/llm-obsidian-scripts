# Session Logs

Daily summaries of Claude sessions.

## Structure

```
Session Logs/
└── YYYY/
    └── MM-Month/
        └── DD/
            └── Session Log YYYY-MM-DD.md
```

## Format

Each exchange is logged as:

```markdown
## HH:MM
**User:** [what was asked]
**Actions:** [tool calls, files touched]
**Response:** [what was accomplished]

---
```

## Notes

- Claude creates these automatically per CLAUDE.md instructions
- Quick way to see what happened in past sessions
- For full transcripts, see raw logs (separate location)
